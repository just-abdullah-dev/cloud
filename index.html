How It Works

We configured the Apache server with the following settings:

  • Enabled Apache2 web server
  • Set proper file permissions (chmod 755)
  • Configured VirtualBox NAT port forwarding (Host 8080 → Guest 80)
  • Tested accessibility from host machine

Apache2 Configuration

Basic Apache configuration for local hosting:

# /etc/apache2/sites-available/000-default.conf
<VirtualHost *:80>
ServerAdmin webmaster@localhost
DocumentRoot /var/www/html

ErrorLog ${APACHE_LOG_DIR}/error.log
CustomLog ${APACHE_LOG_DIR}/access.log combined
</VirtualHost>

Key Commands Used:

sudo systemctl start apache2
sudo systemctl enable apache2
sudo chown -R www-data:www-data /var/www/html/
sudo chmod -R 755 /var/www/html/
